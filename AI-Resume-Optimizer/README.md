# AI Resume Optimizer

## Overview
The AI Resume Optimizer is a prompt-based AI workflow designed to help job seekers improve their resumes and tailor applications to specific job descriptions.

## Problem
Many applicants submit generic resumes that do not highlight the skills and experiences most relevant to a role.

## Solution
I designed an AI-assisted process that:
- Analyzes job descriptions
- Identifies important keywords and skills
- Suggests resume improvements
- Helps create tailored application responses

## Tools Used
- ChatGPT
- Prompt Engineering
- Microsoft Word

## Process
1. Collect job description requirements
2. Analyze key skills and responsibilities
3. Use AI prompts to generate improvements
4. Review and refine outputs
